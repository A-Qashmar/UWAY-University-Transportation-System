# UWAY - University Transportation Web System

## Description
A web-based university transportation system supporting three user roles: Admin, Driver, and Student. This project applies core backend and frontend concepts for managing university transportation.

## Features
- Admin dashboard for managing drivers and students
- Student registration and ride booking
- Driver management system
- Session-based authentication
- Responsive user interface

## Technologies Used
- **Frontend:** HTML, CSS.
- **Backend:** PHP
- **Database:** MySQL
- **Tools:** Git, GitHub, VS Code

## Project Details
- **Duration:** Fall 2025
- **Team Size:** 4 members
- **My Role:** Frontend and Backend Development for Admin dashboard

## What I Learned
- Building responsive admin dashboards
- Session-based authentication systems
- Database design and CRUD operations
- Form handling and input validation
- Team collaboration using Git

## How to Run
1. Clone this repository
2. Set up a local PHP server
3. Create a MySQL database
4. Import the database schema
5. Update database credentials in config file
6. Open in browser at localhost
